MOBILE NUMBER & EMAIL ID PROCESS IN PHONEBANKING

 

When customer calls to enquire the process to change the mobile number or email ID in Credit Cards, PhoneBanker should inform customer on the following options


FORM BASED COM or COE

Customer should send us the mobile number or email ID change form that is available in HDFC Bank website under Form Center. The form should be sent to Chennai Thiruvanmiyur address. Form Name in Form Center is given below

When the form is received the following updates will be done in CBCI by Ops Team.

Mobile number or Email ID updated incorrectly:

 

If customer claims mobile number or email ID is updated incorrectly, PhoneBanker should do the following checks

  • Check the mobile number or email ID mentioned in CBCI & identify the error in the update.
  • Check in the CBCI memo if the below cat & sub cat update is available in the customers card.
CATEGORY SUB - CATEGORY Memo Text
OPS UPDATE WRITTEN INSTRUCTIONS RECEIVED Received your request for   Mobile/Email change on your card ending xxxx. Processing subject to   verification


If the above update is available, PhoneBankers should update the below referral

REFERRAL Mobile   Number issue referral Email   ID issue referral
FILTER ACCOUNT   INFORMATION ACCOUNT   INFORMATION
CATEGORY STATIC   DATA MAINTENANCE STATIC   DATA MAINTENANCE
SUB   CATEGORY MOBILE   NUMBER RELATED EMAIL   ID RELATED


The referral will have the following webform fields

Letter Type: Choose “error in action”

Complaint/Feedback: Update the error that customer mentions in the referral. Do not update the complete mobile number or email ID

On receipt of the referral, Ops team will check the letter received from the customer & make the required changes if the mobile number or email ID is updated incorrectly. PhoneBanking represents customers query to Ops & from thereon any changes will be decided only by Ops Team.

 

Mobile number or Email ID not updated:

If customer claims mobile number or email ID is not updated in credit card account, PhoneBanker should do the following checks

  • Check the date customer sent the request.
  • Check in the CBCI memo if the below cat & sub cat update is available in the customers card.
CATEGORY SUB - CATEGORY Memo Text
OPS UPDATE WRITTEN INSTRUCTIONS RECEIVED Received your request for   Mobile/Email change on your card ending xxxx. Processing subject to   verification


  • If the Ops update is available, bank has received customers letter & the changes will made in the next 3 working days subject to sign verification. If the changes are not made post 3 working days, PhoneBanker should update the below given referral
  • If the Ops update is NOT available after 5 working days of customer sending the letter, PhoneBanker can update the below referral
REFERRAL Mobile   Number issue referral Email   ID issue referral
FILTER ACCOUNT   INFORMATION ACCOUNT   INFORMATION
CATEGORY STATIC   DATA MAINTENANCE STATIC   DATA MAINTENANCE
SUB   CATEGORY MOBILE   NUMBER RELATED EMAIL   ID RELATED


The referral will have the following webform fields

Letter Type: Choose “letter not actioned”

Complaint/Feedback: Update the date customer sent the letter with tracking number if any

On receipt of the referral, Ops team will check if the letter received from the customer & make the required changes.


NETBANKING & E-AGE BASED COM/COE FOR INHOUSE CUSTOMERS

 

Note : Any requests for change in Landline number/email ID/mobile number through NetBanking without a change of address will not be updated in Credit Cards

If customer does not want to change the mobile number or email ID in credit cards and wants to maintain the same mobile number or email ID as per the earlier mobile number or email ID maintained, the below mentioned referral can be raised to back-end team for maintenance.



FILTER: ACCOUNT INFORMATION

CATEGORY: STATIC DATA MAINTENANCE

SUB-CATEGORY: COMBINED ADDRESS CHANGE FORM


Webform Field will need to be selected as YES or NO as per customer revoke request for COM/COE. The case will move to Ops Team who will revoke the mobile number or email ID maintained & update the mobile number or email ID maintained in the credit card again. The above referral can be used to revoke mobile number or email ID changes made through e-age form, NetBanking, combined address change form also to revoke the mobile number or email ID maintained.


RE-KYC BASED MOBILE NUMBER OR E-MAIL ID CHANGE


Bank conducts periodic activity of updating KYC & requests customers to fill the Re-KYC form available in HDFC BANK WEBSITE> FORM CENTRE> FORM - KYC Customer updation. Customer has to attach ID & Address proof along ( List of valid ID & Address proof is given in the form). Customer should send the filled KYC Form with Self attested copy of ID & address proof to Chennai Thiruvanmiyur address or drop the same in a drop box at the branches.


On receipt of the documents, the current customer records will be verified with the Re-KYC proofs now being received & if found incorrect updation will be done in the bank records.


The following updates will be available in CBCI

SDM DEMOGRAPHICS-CRITICAL-EMAIL COE is processed

  • Check in the CBCI memo if the below cat & sub cat update is available in the customers card.

RE-KYC SDM RE-KYC SDM DONE ON RE-KYC DATED   27-DEC-2016

FILTER                   : SERVICE SUPPORT CALLOUTS

CATEGORY         : CENTRAL SS REFERRALS

SUB-CATEGORY: LETTER-ACTION-RELATED


The referral will have the following webform fields

Letter Type: Choose “error in action”

Complaint/Feedback: Update the error that customer mentions in the referral. Do not update the complete address

PhoneBanking represents customers query to Ops & from thereon any changes will be decided only by Ops Team.

If customer is confirming that the mobile number or email ID was correctly mentioned in the credit card Application form (pre-issuance) then the team member will need to rise the below referral


FILTER                   : ACCOUNT INFORMATION

CATEGORY         : APPLICATION RELATED

SUB-CATEGORY: APPLICATION RETRIEVAL


The above referral will queue into the Central PhoneBanking shop.
